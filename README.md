# Example repo

This project is a simple code that prints to the terminal.
The purpose is to demonstrate Git workflow


1. Initializing a repository (`git init`)
2. Staging files (`git add`)
3. Committing changes (`git commit`)
4. Modifying tracked files
5. Pushing to a remote GitHub repository


The program prints an initial "Hello World!" message and then an updated "Git is awesome!" message to show changes.
